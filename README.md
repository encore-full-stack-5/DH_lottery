# 동행복권 사이트 
연금 복권 + 복권 +  스포츠 토토 

## 📃 프로젝트 소개
동행복권 사이트를 모티브로 복권, 연금 복권, 토토, 결제를 구현한 사이트입니다.

게임 당첨 시 당첨 결과를 사용자 이메일로 발신하는 서비스를 구현했습니다.
